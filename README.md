## Требования

- JDK 21+
- IntelliJ IDEA Community
- Git и GitHub аккаунт

---

## Как работать с лабораторной

1. Откройте нужный класс по заданию (см. `docs/ЗАПУСК_КЛАССОВ.md`).
2. Прочитайте Javadoc и блоки `Подсказка`.
3. Реализуйте участки внутри TODO-маркеров:
   - `// ▼ ВАШ КОД ЗДЕСЬ ▼`
   - `// ▲ КОНЕЦ ВАШЕГО КОДА ▲`
4. Запустите `main()` и сравните результат с ожидаемым.
5. Зафиксируйте прогресс коммитом.

---

## Педагогический формат

- В каждом задании есть:
  - краткая цель;
  - пошаговые подсказки без готового решения;
  - мини-набор тестовых значений;
  - раздел `Проверь себя`;
  - раздел `Типичные ошибки`.
- Код шаблонный, но компилируемый: студент добавляет логику по TODO.
